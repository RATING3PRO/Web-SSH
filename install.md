# 快速安装指南

## Windows 用户
1. 双击运行 `start.bat`
2. 等待依赖安装完成
3. 浏览器会自动打开 http://localhost:3000

## 手动安装
```bash
# 安装依赖
npm install

# 启动服务
npm start
```

## 功能说明

### 基础连接
- 支持密码和SSH密钥认证
- SOCKS4/SOCKS5代理支持
- 连接配置保存和管理

### AI助手
1. 点击右上角机器人图标
2. 配置你的AI API（支持OpenAI兼容接口）
3. 开始与AI助手对话，获取Linux命令帮助

### 文件管理
- 切换到"文件管理"标签页
- 浏览、下载、上传、删除文件
- 支持拖拽上传

### 快捷键
- `Ctrl+C` / `Ctrl+V` - 复制粘贴
- `↑` / `↓` - 命令历史
- `Tab` - 自动补全

## 注意事项
- 首次使用需要配置AI API密钥
- 建议在HTTPS环境下使用
- 文件上传支持文本文件
